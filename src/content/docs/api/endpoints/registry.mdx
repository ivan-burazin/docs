Endpoints for managing container registry credentials and operations.

## List Container Registries

`GET /container-registry`

Lists all configured container registries.

### Request

```bash
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.daytona.io/v1/container-registry
```

### Response

```json
{
  "data": {
    "registries": [
      {
        "server": "registry.hub.docker.com",
        "username": "user"
      }
    ]
  }
}
```

## Get Registry Credentials

`GET /container-registry/{server}`

Get credentials for a specific container registry.

### Parameters

| Name | Location | Required | Type | Description |
|------|----------|----------|------|-------------|
| `server` | path | Yes | string | Registry server name |

### Request

```bash
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.daytona.io/v1/container-registry/registry.hub.docker.com
```

### Response

```json
{
  "data": {
    "server": "registry.hub.docker.com",
    "username": "user",
    "authenticated": true
  }
}
```

## Set Registry Credentials

`PUT /container-registry/{server}`

Set credentials for a container registry.

### Parameters

| Name | Location | Required | Type | Description |
|------|----------|----------|------|-------------|
| `server` | path | Yes | string | Registry server name |

### Request Body Parameters

| Name | Required | Type | Description |
|------|----------|------|-------------|
| `username` | Yes | string | Registry username |
| `password` | Yes | string | Registry password/token |

### Request

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "user",
    "password": "pass123"
  }' \
  https://api.daytona.io/v1/container-registry/registry.hub.docker.com
```

### Response

```json
{
  "data": {
    "server": "registry.hub.docker.com",
    "username": "user",
    "authenticated": true
  }
}
```

## Remove Registry Credentials

`DELETE /container-registry/{server}`

Remove credentials for a container registry.

### Parameters

| Name | Location | Required | Type | Description |
|------|----------|----------|------|-------------|
| `server` | path | Yes | string | Registry server name |

### Request

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.daytona.io/v1/container-registry/registry.hub.docker.com
```

### Response

Returns a `204 No Content` response on success.
